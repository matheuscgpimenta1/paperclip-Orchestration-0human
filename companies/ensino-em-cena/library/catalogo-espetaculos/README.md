# Catálogo de Espetáculos

Cada espetáculo em produção ou repertório tem um arquivo próprio aqui.

## Padrão de nome

`{slug-do-espetaculo}.md` — ex.: `o-livro-encantado.md`

## Campos obrigatórios

Use o `_template.md` ao criar um novo espetáculo. Todo arquivo precisa de:

- Título, sinopse, faixa etária, duração
- Ficha técnica (elenco, equipe, requisitos de palco)
- Alinhamento pedagógico (BNCC, temas transversais)
- Materiais associados (encarte, guia do professor)
- Status (em criação, em repertório, suspenso)
- Cachê base (link para `tabela-cache/`)
