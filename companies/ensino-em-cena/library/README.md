# Library — Base de Conhecimento

Repositório central de conhecimento operacional da Ensino em Cena. Estruturas vazias prontas para preenchimento.

## Estrutura

```
library/
├── catalogo-espetaculos/        Cada espetáculo um arquivo, padrão definido
├── catalogo-projetos-leitura/   Cada projeto pedagógico um arquivo
├── tabela-cache/                Tabela de cachês por tipo, raio e segmento
├── bncc/                        Referência de competências e habilidades por etapa
├── brand-kit/                   Identidade visual, vozes, tipografia
├── templates/                   Contratos, propostas, fichas técnicas, releases
└── playbooks/                   Procedimentos repetíveis (onboarding escola, pós-evento)
```

## Princípios

- Tudo aqui é fonte da verdade — agentes consultam, não inventam
- Versionamento por git: cada mudança rastreável
- Esqueletos hoje, preenchimento por Fernanda e especialistas conforme demanda
- Quando um item é citado em proposta/material, link para o arquivo correspondente
