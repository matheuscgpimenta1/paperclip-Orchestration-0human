# Catálogo de Projetos de Leitura e Escrita

Cada projeto pedagógico em portfólio tem um arquivo próprio.

## Padrão de nome

`{slug-do-projeto}.md`

## Campos obrigatórios

Use `_template.md`. Todo arquivo precisa de:

- Título, segmento, duração total, obras envolvidas
- Competências e habilidades BNCC mobilizadas
- Sequência didática (encontros, atividades)
- Materiais (caderno do aluno, guia do professor)
- Avaliação (instrumentos e critérios)
- Status
